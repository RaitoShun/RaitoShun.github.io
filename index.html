Sample Project #2

Facial Recognition Brain

Skills: React, Node.js, CSS3, JS ES8, PostgreSQL, Clarifai API

As the epitome of an excellent Udemy course, this Web App uses a facial recognition API to detect and point out faces. It uses a database to store all the users, as well as their entries. I enjoyed the efficiency of React and operating servers. A chellenge was debugging and streamlining back-end code, back-end security principles and asynchronous functions.

Usage: Login or Register in the app and proceed to supply it with the of URL an online picture.
