ISS Tracker

Skills:

HTML5, THREEJS, CSS3

I love space. I love making webpages. I love 3D effects. I decided to combine the three (See what I did there?) in order to make a real-time International Space Station Tracker. It uses an API I built to gain the coordinates of the ISS , scraped form the NASA website, and converts them to coordinates on Earth. Who knows? Maybe the astronauts are above you right now!

Sample Project #2

Facial Recognition Tracker

Skills:

React, Node.js, PostgreSQL, Clarifai API

This Web App uses a facial recognition API to detect and point out faces. It then uses a PostgreSQL database to store all the users, as well as their entries. I love working with this tech stack the most. Login or Register in the app and proceed to supply it with the of URL an online picture. You'll be surprised how addicting finding faces can get!
