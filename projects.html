Sample Project #1

ISS Tracker

Skills:

HTML5, THREEJS, CSS3

I love space. I love making webpages. I love 3D effects. I decided to combine the three (See what I did there?) in order to make a real-time International Space Station Tracker. It uses an API to gain the coordinates of the ISS at any given time and converts them to be mapped on a 3D model of Earth. Actually having accurate tracking in place was tricky. Who knows? Maybe the astronauts are above you right now!

Sample Project #2

Facial Recognition Tracker

Skills:

React, Node.js, PostgreSQL, Clarifai API

As the epitome of an excellent Udemy course, this Web App uses a facial recognition API to detect and point out faces. It uses a database to store all the users, as well as their entries. I enjoyed the efficiency of React and operating servers. A challenge was debugging and streamlining back-end code, back-end security principles and asynchronous functions.

Usage: Login or Register in the app and proceed to supply it with the of URL an online picture.
